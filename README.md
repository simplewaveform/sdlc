# Аренда автомобилей

Веб-сервис каталога автомобилей (первая итерация: без функционала аренды).

- **Фронт:** React (Vite)
- **Бэк:** Node.js, Express, PostgreSQL

## Запуск

### Бэкенд

1. Установить PostgreSQL, создать БД:
   ```bash
   createdb car_rental
   ```
2. В папке `backend`:
   ```bash
   cp .env.example .env
   # при необходимости отредактировать .env (PORT, DATABASE_URL)
   npm install
   npm run db:init
   npm run db:seed
   npm run dev
   ```
   API: http://localhost:3001

### Фронтенд

В папке `frontend`:

```bash
npm install
npm run dev
```

Приложение: http://localhost:5173 (прокси на API настроен в Vite).

## Первая итерация

- Каталог: сетка карточек (фото, модель, год, цена/сутки BYN), данные с API.
- Детальная страница авто: фото, описание, характеристики (трансмиссия, топливо, объём), кнопка «Забронировать» (заглушка), кнопка «Назад».
- Поиск по модели и фильтр по макс. цене за сутки; сброс фильтров.
