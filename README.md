# Аренда автомобилей

Веб-сервис каталога автомобилей (первая итерация: без функционала аренды).

- **Фронт:** React (Vite)
- **Бэк:** Node.js, Express
- **БД:** SQLite (файл `backend/data/cars.db`) — ничего ставить не нужно. Позже можно заменить на Postgres, поменяв только слой `backend/src/db/`.

## Запуск

**Бэкенд** (в папке `backend`):

```bash
npm install
npm run db:seed
npm run dev
```

Сервер: http://localhost:3001  
При первом запуске таблица создаётся сама. `db:seed` — один раз, чтобы заполнить каталог тестовыми авто.

**Фронтенд** (в папке `frontend`):

```bash
npm install
npm run dev
```

Приложение: http://localhost:5173

## Первый спринт

- Каталог: сетка карточек (фото, модель, год, цена/сутки BYN), данные с API.
- Детальная страница авто: фото, описание, характеристики (трансмиссия, топливо, объём), кнопка «Забронировать» (заглушка), кнопка «Назад».
- Поиск по модели и фильтр по макс. цене за сутки; сброс фильтров.
